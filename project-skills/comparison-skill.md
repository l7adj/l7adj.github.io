# Skill: Company Comparison Article

## Purpose

Use this skill whenever the user asks for a comparison between two companies.

The comparison must be decision-focused, not descriptive only.

## Required Inputs

For both companies collect:

- Company names
- Official URLs
- Program types
- Prices
- Account sizes
- Profit targets
- Daily loss
- Max loss
- Drawdown type
- News trading
- Weekend holding
- EA/bots
- Copy trading
- Hedging
- Payout split
- Payout frequency
- Platforms
- Refund policy
- Main restrictions

## Article Structure

Use this exact structure:

1. H1: [Company A] vs [Company B]
2. Short Verdict
3. Quick Winner Table
4. Full Comparison Table
5. Pricing Comparison
6. Rules Comparison
7. Payout Comparison
8. Platform and Trading Conditions
9. Which Is Better for Beginners?
10. Which Is Better for Scalpers?
11. Which Is Better for News Traders?
12. Which Is Better for Swing Traders?
13. Pros and Cons of Each
14. Final Recommendation
15. FAQ

## Writing Rules

- Do not make both companies equal unless genuinely necessary.
- Give a clear winner for each category.
- Mention uncertainty when data is incomplete.
- Explain the practical meaning of each difference.
- Avoid generic filler.

## SEO Requirements

Title:

`[Company A] vs [Company B] [Year]: مقارنة القواعد، الأسعار، السحب، وأيهما أفضل؟`

Meta description:

`مقارنة شاملة بين [Company A] و[Company B]: الأسعار، القواعد، السحب، التداول وقت الأخبار، المميزات والعيوب، والحكم النهائي.`

Suggested slug:

`comparisons/company-a-vs-company-b.html`
