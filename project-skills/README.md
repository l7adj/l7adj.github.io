# Project Skills - محطة التمويل

هذا المجلد يحتوي على قواعد تشغيل المشروع. هذه ليست صفحات للزوار، بل ملفات مرجعية لتنظيم العمل عند إنشاء أو تعديل محتوى الموقع.

## الهدف

ضمان أن كل مراجعة شركة، مقارنة، صفحة تصنيف، أو تحديث SEO يتم بنفس القالب ونفس المستوى، بدون عشوائية أو اختلاف في الأعمدة أو البنية.

## المهارات المتاحة

1. `company-review-skill.md`  
   قالب مراجعة شركة Prop Firm كاملة.

2. `comparison-skill.md`  
   قالب مقارنة بين شركتين.

3. `seo-adsense-skill.md`  
   قواعد SEO وAdSense والثقة.

4. `data-qa-skill.md`  
   تدقيق الأرقام والقواعد والبيانات.

5. `github-pages-skill.md`  
   قواعد النشر على GitHub Pages بدون كسر الروابط.

## قاعدة العمل

أي محتوى جديد يجب أن يمر بهذه المراحل:

1. جمع البيانات.
2. تنظيمها في نفس القالب.
3. تدقيق الأرقام.
4. كتابة الحكم النهائي.
5. إضافة SEO.
6. التأكد من الروابط.
7. تحديث sitemap عند الحاجة.
